Welcome to pyBrew

pyBrew is a simple, free software for home brewers as an alternative to BeerSmith or similar programs. You can download the source code from GitHub, or get a pre-compiled Windows installer for the latest release

Program Features

  • Create projects to track grain bill, hops used, yeast used, fermentation behavior, and general notes for each of your brewing projects
  • Compare your beer's properties against a number of pre-defined styles

  • Add grains, hops, and yeast to your project from editable databases of ingredients

  • Calculate the tank pressure and line length for your keg system by inputting the refrigerator temperature and desired carbonation

  • Calculate bitterness (IBU) using four different formulas from Dr. Hall's Zymurgy article
